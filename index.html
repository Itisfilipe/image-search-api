Image Search API Created for Free Code Camp

User stories:

  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.

Usage:

https://immense-earth-4759.herokuapp.com/api/imagesearch/cats%20funny?offset=2

Output:

[ { url: "http://i.imgur.com/T9hRUXL.png", snippet: "What happened to just funny shit and cats?", context: "http://imgur.com/T9hRUXL" }, { url: "http://imgur.com/a/LWssZ", snippet: "Cats are funny", context: "http://imgur.com/LWssZ" } ]

Usage:

https://immense-earth-4759.herokuapp.com/api/latest/imagesearch/

Output:

[ { _id: "568c85cc5733aa0300221c89", term: "cats funny", when: "2016-01-06T03:11:08.419Z" }, { _id: "568c85945733aa0300221c88", term: "cats funny", when: "2016-01-06T03:10:12.366Z" }, { _id: "568c85855733aa0300221c87", term: "lolcats funny", when: "2016-01-06T03:09:57.284Z" }, ... ]
